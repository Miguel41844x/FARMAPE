# FARMAPE Documentacion

Esta carpeta queda reservada para documentacion complementaria del backend de microservicios.

Puede usarse para guardar material de apoyo de la exposicion, diagramas de arquitectura, decisiones tecnicas, capturas de despliegue local, evidencia de pruebas o notas de integracion con el frontend.

## Contenido sugerido

```text
docs/
  arquitectura.md
  despliegue-docker.md
  despliegue-kubernetes.md
  pruebas-manuales.md
```

La documentacion principal del proyecto se mantiene en `backend-microservices/README.md`; esta carpeta sirve como extension cuando se requiera mayor detalle.
